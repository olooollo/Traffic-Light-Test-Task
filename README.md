## Что сделано
- **Модели**:
  - `Role`: `name`, `default_salary`
  - `Department`: дерево
  - `Employee`: `full_name`, `role`, `employment_date`, `salary`, `department`
- **Главная страница** (`/`): выводит **дерево департаментов** с возможностью раскрытия узлов.
  - Дерево отображает иерархию департаментов с указанием количества работников
  - UI на Twitter Bootstrap (Collapse)
- **Seeder**: management-command для заполнения БД:
  - 25 департаментов
  - 5 уровней иерархии
  - 60 000 сотрудников
  - Несколько ролей (по умолчанию 10)
  - Генерация случайных ФИО

### Заполнение базы данными (25 департаментов / 5 уровней / 60 000 сотрудников)
Команда заполнит БД данными:

```bash
python manage.py tree_init --truncate
```

## Примечания
* Для больших деревьев можно использовать ленивую подгрузку, но для размеров дерева в ТЗ решил использовать один запрос
